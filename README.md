# metricas
Repositorio usado para a aula de métricas de software
=======
## Como usar

1. Clonar a aplicação
```shell
git clone https://github.com/estacioceara/sistema-escola-natacao
```

2. Entrar no diretório
```shell
cd sistema-escola-natacao
```

3. Subir o docker
```shell
make setup
```

Tá pronto o solvetinho, deve estar rodando em <http://localhost:8080>
